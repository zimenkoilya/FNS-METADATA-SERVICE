# FNS-metadata-service


## How to setup

```
git clone https://github.com/zimenkoilya/FNS-METADATA-SERVICE
cd FNS-METADATA-SERVICE
cp .env.org .env // Fill in Vars
yarn
yarn dev
```


## How to deploy

```
yarn deploy
```


## How to test

Regular unit test;
```
yarn test
```

Unit test + coverage;
```
yarn test:cov
```
